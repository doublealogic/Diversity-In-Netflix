How Has Netflix Changed?

Perception is changing in the way we watch movies and TV, and we wanted to see how those changes reflect in our data that we collect.

Questions we asked:

  • Which country has the most representation on Netflix?
  • Which genres (“listed_in”) have the most representation. Which has the least?
  • Who is winning the war between platform dominance: movies or TV series?

Where we acquired data:

  • Kaggle dataset Netflix Movies and TV Shows. Click Here for the website

    This dataset has movies and tv shows available on Netflix as of the end of 2019. The creation of the dataset has similar interest to our project. We used postgres to host the database.

  • Trakt API Click Here for the website documentation

    Trakt api collects data on what TV shows and movies people are watching. We were able to pull data and merge it with our Netflix dataset.
